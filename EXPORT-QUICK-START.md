# 🚀 导出功能快速上手

## 📥 如何使用新的导出功能？

### 1️⃣ 打开高级导出

在管理后台点击 **"📥 高级导出"** 按钮

### 2️⃣ 选择导出方式

点击其中一个按钮：

| 按钮 | 用途 | 何时使用 |
|------|------|----------|
| 📊 **Excel** | 导出完整表格 | 需要编辑、分析数据 |
| 📄 **CSV** | 导出纯文本 | 需要导入其他系统 |
| 🖨️ **打印** | 直接打印 | 需要纸质报表 |
| 📋 **剪贴板** | 快速复制 | 需要粘贴到其他地方 |

### 3️⃣ （可选）设置筛选条件

如果需要筛选数据：

- **时间范围**：选择起止日期
- **地区**：选择省/市/区
- **状态**：选择订单状态
- **关系**：选择客户关系

### 4️⃣ 查看预览并导出

- 查看 **"将要导出"** 的记录数
- 确认无误后点击导出方式
- 等待"导出成功"提示

---

## 💡 常用场景

### 场景1：导出所有数据

```
操作：直接点击"Excel"或"CSV"
结果：导出全部记录
```

### 场景2：导出昨天的订单

```
操作：
1. 时间范围：起始和结束都选昨天日期
2. 点击"Excel"

结果：只导出昨天的记录
```

### 场景3：导出北京地区待发货订单

```
操作：
1. 地区：选择"北京市"
2. 状态：选择"待发货"
3. 点击"Excel"

结果：北京待发货的订单清单
```

### 场景4：快速复制客户信息

```
操作：
1. 筛选出需要的记录
2. 点击"剪贴板"
3. 粘贴到记事本

结果：可直接查看或打电话核对
```

## 📊 查看导出记录

点击 **"📊 导出记录"** 按钮查看最近5次导出：

- 导出时间
- 文件名
- 记录数量
- 文件大小
- 筛选条件

---

## ⌨️ 快捷技巧

1. **重置筛选**：点击"🔄 重置筛选"清空所有条件
2. **预览数量**：设置筛选后立即显示符合条件的记录数
3. **清空记录**：在导出记录界面点击"🗑️ 清空记录"
