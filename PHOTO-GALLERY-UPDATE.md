## 🚀 技术实现细节

### 无限循环原理
```html
<!-- 相册包含两组相同的照片 -->
<div class="gallery-track">
    <!-- 第一组：8张照片 -->
    <div class="gallery-item">...</div> <!-- photo1 -->
    ...
    <div class="gallery-item">...</div> <!-- photo8 -->
    
    <!-- 第二组：相同的8张照片 -->
    <div class="gallery-item">...</div> <!-- photo1 -->
    ...
    <div class="gallery-item">...</div> <!-- photo8 -->
</div>
```

### CSS 动画
```css
.gallery-track {
    display: flex;
    gap: 20px;
    animation: autoScroll 40s linear infinite;
}

@keyframes autoScroll {
    0% {
        transform: translateX(0);
    }
    100% {
        transform: translateX(-50%); /* 移动到第二组的起点 */
    }
}
```

**原理说明**：
- `translateX(-50%)` 表示向左移动总宽度的 50%
- 当第一组照片滚动完毕时，第二组照片正好接上
- 由于两组照片完全相同，视觉上形成无缝循环

### 触摸交互
```javascript
// 触摸开始：暂停动画
track.style.animationPlayState = 'paused';

// 触摸结束：恢复动画
track.style.animationPlayState = 'running';
```

## 📊 性能优化

1. **硬件加速**
   ```css
   .gallery-track {
       will-change: transform;
   }
   ```

2. **图片懒加载**
   ```html
   <img src="..." loading="lazy">
   ```

3. **合理的动画时长**
   - 40秒完成一轮（8张照片 × 5秒/张）
   - 避免过快导致眼花
   - 避免过慢导致无聊
