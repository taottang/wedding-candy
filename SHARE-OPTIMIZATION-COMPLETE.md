### 3. 精美海报样式 🎨

#### 视觉优化细节

##### 整体风格
```
Before（旧版）：
- 简单边框
- 单一背景
- 无装饰

After（新版）：
- 渐变背景（3色过渡）
- 圆形光晕装饰
- 脉动动画
- 多层阴影
- 文字渐变
```

##### 具体优化

**1. 背景设计**
```css
/* 渐变背景 */
background: linear-gradient(
    135deg, 
    #FAF8F5 0%, 
    #F5F0EB 50%, 
    #FAF8F5 100%
);

/* 圆形光晕装饰 */
::before, ::after {
    radial-gradient(
        circle,
        rgba(216, 180, 180, 0.15) 0%,
        transparent 70%
    );
}
```

**2. 新人姓名**
```css
/* 优化前 */
font-size: 48px;
letter-spacing: 0;

/* 优化后 */
font-size: 52px;          /* 加大 */
letter-spacing: 8px;      /* 增加字间距 */
text-shadow: 2px 2px 4px rgba(...); /* 添加阴影 */
```

**3. 脉动动画**
```css
@keyframes pulse {
    0%, 100% { transform: scale(1); }
    50% { transform: scale(1.05); }
}

.poster-icon {
    animation: pulse 2s ease-in-out infinite;
}
```

**4. 内容卡片**
```css
/* 优化 */
- 增强阴影（0 8px 30px）
- 右上角心形装饰（半透明）
- 标题下划线 + 渐变装饰
- 文字行高加大（2）
```

**5. 高亮信息框**
```css
/* 优化前 */
background: linear-gradient(120deg, #D8B4B4 0%, #E8DDD4 100%);
border-left: 4px solid #D8B4B4;

/* 优化后 */
background: linear-gradient(
    135deg, 
    rgba(216, 180, 180, 0.3) 0%, 
    rgba(232, 221, 212, 0.3) 100%
); /* 降低透明度，更柔和 */
border: 2px solid rgba(216, 180, 180, 0.5);
border-left: 5px solid #D8B4B4; /* 加粗左边框 */
box-shadow: 0 4px 15px rgba(216, 180, 180, 0.15); /* 添加阴影 */
font-weight: 500; /* 加粗字体 */
```

**6. 二维码区域**
```css
/* 优化 */
- 边框加粗（3px → 4px）
- 阴影增强并带粉色调
- "长按识别"文字渐变效果
- 整体区域添加淡淡渐变背景
```

**7. 文字渐变效果**
```css
.poster-qr-hint {
    background: linear-gradient(135deg, #D8B4B4, #C89FA0);
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
    background-clip: text;
}
```
